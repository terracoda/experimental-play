PhET About Dialog in HTML5 with ARIA

Presented here are two samples of the About Dialog for the Molecule Shapes Simulation. The first sample presents the contents of the dialog as it is currently laid out in the live simulation. In the second sample, I have re-arranged the content slightly by moving the PhET copyright information to the footer and changing the Credit heading to Simulation Design Team. I think the re-arrangement works better if landmarks are to be used in the dialog.

The Molecular Shapes simulation is used as the example, as it is the most recently published simulation.

After each visual concept, I present the code with added aria attributes.

Note the aria document role, role="document" is needed if content in the dialog is to be made available to the reading keys of a screen reader. Without the document role, the only content that is read out is label and description information, and focusable items.

Simple About Dialog (no landmarks)

Content of the About Dialog as-is in the Molecule Shapes Simulation.

HTML5 & ARIA Code (no landmarks)

<section role="dialog" aria-labelledby="dialogTitle" aria-describedby="dialogDesc">
<button aria-label="Close" href="#">X</button> 
<h1 id="dialogTitle">Molecule Shapes</h1>
  <ul id="dialogDesc">
    […version information]
  </ul>
  <div role="document">
    <h2>PhET<sup>&reg;</sup> Interactive Simulations</h2>
      <p>Copyright &copy; 2004-2015 University Colorado Boulder</p>

  <h2>Credits</h2>
    <ul>
      […credit information];
    </ul>

    <ul>
      […link to terms, licensing, third party credits]
    </ul>
  </div> <!-- end document -->
</section> <!-- end dialog -->

About Dialog with Landmarks

Contents of the About Dialog re-arranged slightly.

About Dialog Code with Landmarks

<section role="dialog" aria-labelledby="dialogTitle2" aria-describedby="dialogDesc2">
  <header>
    <button type="button" aria-label="Close">X</button>
    <h1 id="dialogTitle2">Molecule Shapes</h1>
    <ul id="dialogDesc2">
      […version information] 
    </ul>
  </header>

  <div role="document">
    <h2 aria-describedby="creditDesc">Credits</h2>
    <ul id="creditDesc">
      […credit information]
    </ul>
  </div> <!-- end document role -->

  <footer aria-labelledby="copyrightTitle" aria-describedby="copyrightDesc">
    <h2 id="copyrightTitle">PhET<sup>&reg;</sup> Interactive Simulations</h2>
    <p id="copyrightDesc">Copyright &copy; 2004-2015 University Colorado Boulder</p>
    <ul>
      […link to terms, licensing, third party credits] 
    </ul>
  </footer>
</section> <!-- end dialog -->

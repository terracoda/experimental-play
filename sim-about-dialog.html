Sample Layout and Code for the About Page Dialog

Below are two samples of the About page for the Molecule Shapes Simulation. The first sample is content as it is currently laid out in the live simulation, but with ARIA attributes added. The second sample, adds landmark tags and moves the PhET copyright information to the footer.

Note the aria document role, role="document" makes the content that content in dialog available to the reading keys of a screen reader.

Code for Simple Dialog

<section role="dialog" aria-labelledby="dialogTitle" aria-describedby="dialogDesc">
<button aria-label="Close" href="#">X</button> 
<h1 id="dialogTitle">Molecule Shapes</h1>
  <ul id="dialogDesc">
    […version information]
  </ul>
  <div role="document">
    <h2>PhET<sup>&reg;</sup> Interactive Simulations</h2>
      <p>Copyright &copy; 2004-2015 University Colorado Boulder</p>
  
  <h2>Credits</h2>
    <ul>
      […credit information];
    </ul>
  
    <ul>
      […link to terms, licensing, third party credits]
    </ul>
  </div> <!-- end document -->
</section> <!-- end dialog -->
